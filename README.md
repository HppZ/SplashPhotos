# SplashPhotos

这是一个图片App，片源来自[https://unsplash.com/][1]，可浏览最新照片，分类照片，集合浏览，集合详细，用户信息，大图查看，下载原图，原图查看等。

我的侧滑菜单：[HPSideMenu][2]
